    Welcome to CMS.

    This php-based content management system doesn't have a title yet, but see what we have done:

    • [x] language support (english texts will get translatet due to function _t to selectet lang)
    • [x] action handling:
      1. plug-ins can queue their functions to actions
      2. advanced parameter passing and callback possibilitys
    • [x] theme / widget frontend core
    • [x] caching system
    • [ ] administrator editing functionality:
      1. creating new sites
      2. changing sites
      3. editing widgets /themes
      4. widget / theme /plug-in overview and control
      5. user control

    The goal

    We want to create with this CMS a perfect prepared system with all widgets / themes and plug-ins a person needs, but give them the opportunity to change everything the way they want it. The system should be easy to use for normal internet user but flexible for nerds to mod it.
